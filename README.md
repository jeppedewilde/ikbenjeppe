# ikbenjeppe

## Week 1
### Woensdag 4 februari
Wat heb ik vandaag gedaan? > workshop api 1 en 2, beginnetje gemaakt website, gekloot met javascript om animatie aan de praat te krijgen.

Hoelang duurde het? > workshops duurden bij elkaar 2,5 uur, de rest van de dag gebruikt voor prutsen website.

Wat heb ik vandaag geleerd? > hoe api ongeveer werkt (basics)

Wat ga ik morgen doen? > zorgen dat ik een werkende animatie heb.


### Donderdag 5 februari
Wat heb ik vandaag gedaan? > light/dark mode workshop, light/dark mode toegevoegd aan website, css animatie luchtbellen, zorgen dat als je over de luchtbellen hovered deze poppen en als alle luchtbellen gepopt -> button verschijnt.

Hoelang duurde het? > workshop duurde 1,5 uur, daarna 4,5 uur gewerkt aan bovenstaande features.

Wat heb ik vandaag geleerd? > dat light/dark mode helemaal niet zo ingewikkeld is, kennis gerefreshed over css animations met keyframes en js functions.

Wat ga ik morgen doen? > zorgen dat er iets gebeurd bij klikken op de dive knop (iets van een animatie/nieuwe fase van de website die je dan in gaat).


### Vrijdag 6 februari
Wat heb ik vandaag gedaan? > Aan kleine details gewerkt zoals floating diver en het vertonen van meer informatie over mij uit de API.

Hoelang duurde het? > Hier ben ik enkele uren mee bezig geweest.

Wat heb ik vandaag geleerd? > Beter leren omgaan met informatie uit de API halen.

Wat ga ik maandag doen? > Zorgen dat de slide show met foto's werkt en meer detail toevoegen.


### Voortgang week 1
Ik kwam deze eerste week wat moeilijk op gang. Ik kreeg het idee voor mijn website niet helemaal uitgedacht in mijn hoofd en zag er na dag 2 eigenlijk geen toekomst in. Daarom besloot ik mijn idee helemaal om te gooien. Van een vaag plan om "iets" te doen met een sportschool of een sportend figuurtje ging ik naar een redelijk uitbedacht plan: de website vormgeven als de ervaring van een duik, waarbij de bezoeker bij dieper duiken tot meer informatie over mij komt. Toen ik een duidelijker idee had kwam de code ook sneller, hoewel ik wel merkte dat het na ruim 9 maanden niet coderen echt even flink inkomen was. Aan het eind van de eerste week had ik een begin staan met de duiker, de luchtbellen en een beginnetje voor de achterkant van de card.


## Week 2
### Maandag 9 februari
Wat heb ik vandaag gedaan? > Ik was ziek thuis maar heb toch mn best gedaan: slide show met foto's werkend gemaakt, golven animatie op de voorkant van de card, golvende text gemaakt, geprobeerd het onderwaterdeel onder de golven een andere kleur te geven.

Hoelang duurde het? > Ik heb zo'n 5 uur achter mijn laptop gezeten.

Wat heb ik vandaag geleerd? > Horizontal scroll maken.

Wat ga ik morgen doen? > Onderwater een andere kleur geven dan boven water.


### Dinsdag 10 februari
Wat heb ik vandaag gedaan? > workshop dry code van cyd, onderwater onder de golven moving gradient background, slide toegevoegd met favorite movies: mijn favorite movie en random fav movies van klasgenoten (uit API), meer foto's toegevoegd aan de slide show met foto's, emoji's die iets over mij zeggen in de luchtbellen, animatie luchtbellen mooier gemaakt met scale, mini diver image op de slides, begin leerdoelen slide.

Hoelang duurde het? > de workshop duurde 1 uur, de rest van de dag (7 uur) ben ik bezig geweest met de bovenstaande features.

Wat heb ik vandaag geleerd? > refresher op hoe math random werkt, hoe je utility classes kan gebruiken voor nettere code.

Wat ga ik morgen doen? > leerdoelen slide afmaken en de hele website responsive maken.


### Woensdag 11 februari
Wat heb ik vandaag gedaan? > Code review met alexia, leerdoelen slide afgemaakt, gewerkt aan responsiveness, luchtbellen animatie verbeterd, diver img vervangen door foto met met betere kwaliteit, kleine dingetjes toegevoegd zoals een favicon en text animation on hover.

Hoelang duurde het? > Start vd dag & code review duurde 45 min, overige 6/7 uur gewerkt aan de rest van bovenstaande features.

Wat heb ik vandaag geleerd? > Hoe ik een animatie minder zwaar kan maken voor de browser.

Wat ga ik morgen doen? > Afsluiting sprint 0.


### Voortgang week 2
De tweede week begon wederom enigszin moeizaam omdat de griep mij te pakken had gekregen, maar ik merkte dat het coderen zelf steeds natuurlijker kwam. Ik werkte aan vloeiende animaties en nieuwe, ingewikkelde javascript code. Ik heb deze week veel toegevoegd en ik ben best wel een beetje trots op wat ik uiteindelijk heb neergezet.
